Follow these steps to run the project locally:
---

## 1. Clone the repository

```bash
git clone https://github.com/aryemv/images-fetching-project.git
cd images-fetching-project
```

## 2. Install dependencies

Open **two terminals** in the root folder.

### In the first terminal (server):

```bash
cd server
npm install
```

### In the second terminal (client):

```bash
cd client
npm install
```

## 3. Run the project

### In the **server** terminal:

```bash
node server.js
```

### In the **client** terminal:

```bash
npm run dev
```

## 4. Open the app

Visit:

```
http://localhost:5173
```

The React app will be live and connected to your Express backend.
